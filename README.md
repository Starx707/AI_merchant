# AI_merchant
A project where a knowledgeable AI can answer all you questions about the legend of Zelda! 
As of 24/04, upload date, not finished yet

# Installation requirements
To have this project run on your device there are some things you have to install
- install express
- install @langchain/openai @langchain/core [https://docs.langchain.com/oss/javascript/langchain/overview]
- (Be sure to change "type":"model" in your package.json file!)
- install cors
- All done!

# Other requirements
In this project I've worked with the Azure Language model. This requires you to take care of 
your own API key to be able to use the model I'm using.

Once you got this key be sure to add this within a .env file to make the project work!

> Then you're all set!
